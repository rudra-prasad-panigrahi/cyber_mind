# 🛡️ AI Cyber Defense Platform

## 🚀 Description
An advanced AI-based cybersecurity platform that monitors, analyzes, and predicts cyber attacks in real time. It provides intelligent threat detection and interactive dashboards for better security insights.

## 🧠 Features
- Real-time cyber attack monitoring  
- AI-based threat prediction  
- Interactive dashboard with analytics  
- Attack visualization system  
- Backend data processing

### Frontend
- React
- Tailwind CSS
- Chart.js

### Backend
- Node.js
- Python (Flask / FastAPI)

- Scikit-learn
- Pandas
- NumPy
